# MVP Scope

## Objective

Prove the platform can:

- Model student understanding
- Detect misconceptions
- Adapt learning paths

---

## Core Domain

Fractions (deep modeling)

Concepts:

- Understanding fractions
- Equivalent fractions
- Simplifying fractions
- Comparing fractions
- Fraction addition
- Fraction subtraction
- Fraction word problems

Micro-skills:

~25–35

Misconceptions:

~8–12

---

## Additional Topics

Light coverage:

- Decimals
- Percentages
- Ratios

---

## MVP Features

- Knowledge graph
- Micro-skill mastery tracking
- Adaptive learning loop
- Misconception detection
- Practice generation
- Guided learning UI

---

## Excluded From MVP

- Animated avatars
- Parent dashboards
- School integrations
- Gamification
- Multi-subject engine
- Bayesian models
